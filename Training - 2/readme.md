TypeScript
==============
* Angular code can be writte - Dart, Typescript or Javascript
* Typescript introduces a concept of "types".
* There are interfaces - define a state, classes.
* Typescript transpiles to js.
* Typescript version = tsc -v
* Transpiling Code = tsc | Run it in the tsconfig path
* Based on the configuration in tsconfig.json file, the output directory is created.
* Map file is useful for debugging.
